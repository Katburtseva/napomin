# Простая Telegram-напоминалка

Минимальный бот на Python, который умеет создавать напоминания командой:

```text
/remind <ЧЧ:ММ> <текст>
```

Пример:

```text
/remind 19:30 выключить чайник
```

## Что умеет

- `/start` - показывает приветствие
- `/help` - показывает инструкцию
- `/remind 10:00 купить молоко` - отправит сообщение в 10:00 по Москве

## Как запустить

1. Создай бота через `@BotFather` и получи токен.
2. Установи Python 3.11+.
3. Установи зависимости:

```bash
pip install -r requirements.txt
```

4. Задай переменную окружения с токеном.

PowerShell:

```powershell
$env:TELEGRAM_BOT_TOKEN="сюда_твой_токен"
```

5. Запусти бота:

```bash
python bot.py
```

## Важно

- Время всегда интерпретируется по часовому поясу Москвы.
- Если указанное время уже прошло сегодня, напоминание придет завтра в это же время.
- Напоминания хранятся только в памяти.
- Если бот перезапустится, уже созданные напоминания пропадут.
- Это самый простой учебный вариант, без базы данных и без поддержки естественного языка.
